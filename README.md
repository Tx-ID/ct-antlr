# ct-antlr

Will convert grammar from `SimpleLISP.g4` file to minimalist LISP.

Example: 
> `jadi plis buka plis tambah plis buka plis kurang 70 69 plis tutup 1337 plis tutup`

Output:
> `( + ( - 70 69 ) 1337 )`

- Basic terminal knowledge
- A functional brain.
